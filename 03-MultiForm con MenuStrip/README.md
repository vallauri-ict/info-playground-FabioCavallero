# info-playground-fabio-cavallero

_Stud. Fabio Cavallero_

###### MultiForm con menu strip

Progetto Windows Form con Form MDI che presenta tutte le funzionalità svolte 
nel primo esercizio fatto in classe. Al posto dei pulsanti usati per aprire le finestre
sono state usate apposite voci di menù. Da gestire anche la voce di menù Esci. 
E' stato inserito un toolStrip che contenga l'informazione di quali voci di menù 
sono state premute.
